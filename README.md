Learning a CartPole environment using the InvertedPendulum-v5 from MuJoCo. 
To download all necessary libraries run in terinal:

pip install torch numpy gymnasium[mujoco] matplotlib tqdm

Then run demonstrate.py to see the Cart pole in action from the trained policy. Edit hyperparameters.py to change any hyperparameters and run training_saving.py to save the trained models in the subfolders.
